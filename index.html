Project

Howdz 🔸 浏览器起始页

⚡Github📖Document💾Online

完全自定义配置的浏览器导航首页面板，支持自定义添加天气、时钟、搜索、收藏等各种物料组件，可编辑删除组件与其各种样式或功能属性。响应式设计，可自定义随机壁纸背景图或动态壁纸。

安装方式
  • 网页版 - View Online
  • Chrome插件 - Install from store
  • Edge插件 - Install from store
  • 桌面应用 - Download install package

Howdyjs 🔸 插件库

⚡Github📖Document💾NPM

个人JS插件库，包含自定义拖拽、滚动条、右键菜单、图片放大等原生JS插件和动画模态框、标签页等Vue3组件或指令。

预览组件
  • Animation Dialog

    使用动画打开模态框

  • Resize

    为HTML盒子加入拖拽更改大小功能

  • Scroll

    为HTML盒子加入自定义滚动条

  • Size Observer

    监听盒子大小变化

  • Img Zoom

    为Img标签加入图片放大功能，支持按组浏览下一张图片

  • To Drag

    为Fixed定位的元素加入可拖拽功能

  • Standard Tabs

    移动端标签页

  • Mouse Menu

    自定义右键菜单

  • To Control

    将元素设置为可拖拽改变位置与大小
